# VlinderImages

Repository voor opgeslagen tekeningen en gegenereerde 3D modellen van "De Vlindertuin" applicatie.

## Inhoud

- **uploads/**: Opgeslagen tekeningen van gebruikers (PNG format)
- **models/**: Gegenereerde 3D modellen (OBJ en GLB format)

## Gebruik

Deze bestanden worden automatisch gegenereerd door de "De Vlindertuin" applicatie wanneer gebruikers tekeningen maken en naar 3D converteren.
